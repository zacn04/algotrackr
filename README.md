## Automated Setup

1. **Ensure Prerequisites:**
   - **PostgreSQL:** Ensure PostgreSQL is installed on your machine. 

2. **Clone the Repository:**
   ```bash
   git clone https://github.com/zacn04/leettrack.git
   cd leettrack
   ```
3. **Run Setup Script:**
   ```bash
   bash setup.sh
   ```
